# Лабораторна робота №1  
## Розробка застосунку “Форма опитування” (WPF)

### Мета роботи
Ознайомлення з технологією **Windows Presentation Foundation (WPF)** та створення сучасного графічного застосунку для проведення опитування користувача.

---

## Технології
- C#
- .NET / .NET Framework
- Windows Presentation Foundation (WPF)
- XAML
- Visual Studio

---

## Опис застосунку

Застосунок реалізує:

- Введення імені користувача
- Введення email
- Перевірку коректності введених даних
- Перехід між вікнами
- Введення текстового відгуку
- Збереження інформації у файл

---

## Архітектура

Проєкт складається з двох вікон:

### MainWindow
- Введення імені
- Введення email
- Валідація
- Активація кнопки при коректних даних

### SecondWindow
- Відображення введених даних
- Поле для введення відгуку
- Кнопка збереження у файл
- Повернення до головного вікна

---
